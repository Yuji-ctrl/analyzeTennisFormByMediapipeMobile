# form_analyse

A new Flutter project.
MediaPipeを用いてスポーツの動作を分析します。第1弾としてテニスのサーブを取り上げますが、時間があればエアKやボレー、野球のピッチングやサッカーのシュートなどテニスの他動作や他競技の動作を分析できるモードも実装するつもりです。

解析結果はお手本動画との類似度を数値で算出するつもりですが、今後の展望としてはAPIの生成AIを利用して文章での分析レポート等の生成にも取り組みます。
